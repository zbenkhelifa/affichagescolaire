AffichageScolaire centralise toutes les informations utiles de votre établissement dans une interface unique, pilotée en temps réel depuis n'importe quel appareil.

🚫
Professeurs absents
Affichez les absences avec date, matière, classes concernées et type de remplacement. Badge "Aujourd'hui" pour les absences du jour, date précise pour les absences à venir.

Conseils de classe
Planning des conseils par trimestre. Chaque entrée affiche la classe, la date, l'heure et la salle. Filtrés automatiquement pour n'afficher que les dates à venir.
Affichage + Admin

Sorties scolaires
Voyages et sorties pédagogiques avec destination, classes, accompagnateur et heure de départ. S'affichent jusqu'à la date de la sortie.
Affichage + Admin

Événements

Accéder à l'admin
Cliquez sur le bouton ⚙ Admin en haut à droite de l'écran d'affichage. Saisissez le mot de passe (défaut : ecole2024).

Naviguer entre les sections

Section absences

Profs absents

Signalez rapidement une absence avec toutes les informations utiles. Les élèves et les parents voient immédiatement qui est absent, quelle matière est concernée et comment le cours est géré.

Comment ajouter une absence

1
Dans l'admin, ouvrez 🚫 Profs absents
2
Renseignez Nom du professeur et Date (obligatoires)
3
Ajoutez optionnellement la matière, les classes et le type de remplacement (Annulé / Remplacé / Travail à faire…)
4
Cliquez ➕ Ajouter — l'absence apparaît dans le tableau et sur l'écran au prochain rafraîchissement
🍽
Menu cantine
Menus de la semaine
  • Fenêtre glissante de 14 jours
  • Entrée, plat, accompagnement, fromage, dessert
  • Jour de la semaine calculé précisément
  • Modification inline directement dans le tableau
Section cantine

Menu cantine

Renseignez les menus de la semaine une fois, ils s'affichent automatiquement sur l'écran. L'application n'affiche que les menus des 14 prochains jours.

Comment ajouter un menu

1
Ouvrez 🍽 Menu cantine dans l'admin
2
Sélectionnez la date du repas
3
Remplissez les champs : entrée, plat, accompagnement, fromage, dessert (tous optionnels sauf la date)
4
Pour modifier un menu existant, cliquez sur ✏️ dans le tableau — les champs deviennent éditables en place
📢
Bandeau défilant
Messages permanents
  • Plusieurs messages en rotation continue
  • 4 vitesses : 🐢 Lent · ▶ Normal · ⏩ Rapide · 🚀 Très rapide
  • Taille de texte S / M / L / XL
  • Vitesse et taille sauvegardées en base
Section bandeau

Bandeau d'informations

Le bandeau doré en bas d'écran défile en permanence. Idéal pour les annonces récurrentes ou urgentes que tout le monde doit voir.

Comment gérer le bandeau

1
Ouvrez 📢 Bandeau d'infos dans l'admin
2
Tapez votre message et cliquez ➕ Ajouter
3
Ajustez la taille du texte (S/M/L/XL) et la vitesse (🐢/▶/⏩/🚀) avec les boutons en haut du panel
4
Cliquez ✏️ pour modifier un message existant sans le supprimer et recréer

Section établissement

Réglages établissement

Personnalisez l'identité visuelle de votre tableau d'affichage et choisissez le thème qui correspond à l'ambiance de votre établissement.

Configuration initiale recommandée

1
Ouvrez 🏫 Établissement dans l'admin
2
Renseignez le nom de l'établissement — il s'affiche en temps réel dans l'aperçu
3
Uploadez votre logo (JPG/PNG/SVG, max 2 Mo) ou choisissez un emoji représentatif
4
Choisissez un thème dans la section 🎨 — les changements sont immédiats sur l'écran
5
Cliquez 💾 Enregistrer pour sauvegarder toutes les modifications

Personnalisation

Trois thèmes pour
chaque ambiance

Changez l'apparence complète de l'écran d'affichage depuis l'onglet Établissement. Le changement est immédiat et persisté en base de données.

🌙 Thème sombre
Bleu nuit profond, accents dorés. Idéal pour les couloirs sombres ou les TV éloignées des fenêtres. Thème par défaut.
☀️ Thème clair
Fond blanc cassé, textes sombres. Parfait pour les espaces lumineux, les halls d'entrée baignés de lumière naturelle.
🎨 Couleur personnalisée
7 presets (forêt, bordeaux, marine, violet, bois, teal, anthracite) ou n'importe quelle couleur via le sélecteur libre.

n'importe quel écran

Smart TV
Samsung, LG, Sony — tout écran avec navigateur intégré Chrome ou WebKit
🔌
Chromecast / Fire Stick
Projetez l'onglet depuis un PC ou utilisez le navigateur intégré
🫐
Raspberry Pi
Mode kiosk avec Chromium — solution autonome et économique
📱
Tablette / Mobile
Interface responsive — admin accessible depuis smartphone ou tablette
